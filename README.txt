Парсер данных HeadHunter.

Для поиска вакансий перейдите на страницу "Ввод данных" и введите поисковый запрос в форме на странице.
В запросе можно указывать несколько несколько критериев поиска (наименования навыков и места вакансий),
связанных логическими операторами OR и/или AND.

Например:   (Java OR Pyton) AND Django
            Python AND Москва

После отправки закпроса кнопкой "Искать вакансии" и ответа сервера HH, результаты будут выведены на
странице "Результаты". Если перейти на страницу "Результаты" без ввода поискового запроса, на странице будут
показаны результаты последнего запроса.

Страница "Контакты" - демонстрационная, никаких операций не производит.
