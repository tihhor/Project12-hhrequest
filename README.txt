Парсер данных HeadHunter.

Для поиска вакансий перейдите на страницу "Ввод данных" и введите поисковый запрос в форме на странице.
В запросе можно указывать несколько несколько критериев поиска (наименования навыков и места вакансий),
связанных логическими операторами OR и/или AND.

Например:   (Java OR Pyton) AND Django
            Python AND Москва

После отправки закпроса кнопкой "Искать вакансии" проверяется, есть ли в базе данных результаты такого же запроса,
сделанного ранее. Если есть, то запрос на сервер HH не выполняется, на страницу "Результаты" выводятся полученная
ранее информация (с указанием даты и времени получения).

Если раньше такого запроса не было, делается запрос на сервер HH, результаты сохраняются в базу данных
hh_requests.sqlite и выводятся настранице "Результаты". Если перейти на страницу "Результаты" без ввода
поискового запроса, на странице будут показаны результаты последнего запроса, сохраненного в базе данных.

Страница "Контакты" - демонстрационная, никаких операций не производит.
